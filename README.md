## Lokális telepítés, tesztelés, futtatás

### 1. Környezet

A program futtatásához szükséges a Node.js és a NPM telepítése. Valamint a MongoDB szerver elérhetősége. (én lokális MongoDB-t használtam)

### 2. Telepítés

a `backend` mappában:

```bash
npm install
```

a `frontend` mappában:

```bash
npm install
```

### 3. Futtatás

a `frontend` mappában:

```bash
npm run build
```

majd a `backend` mappában:

```bash
npm start
```
